# Trench-Track
Juego de estrategia bélica para un proyecto de historia.

## Ambientación
La partida comienza en la **Primera Guerra Mundial**: el mapa está dominado por trincheras y el cielo surcado por biplanos. A medida que avanza la partida, la tecnología evoluciona y se hace la transición a la **Segunda Guerra Mundial**, donde los tanques y los bombarderos toman el protagonismo en el campo de batalla.
